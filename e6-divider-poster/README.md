# e6-divider-poster

E6系列の抵抗2本（R1-R2）だけで実現できる2直列抵抗の分圧比を、1%〜50%の範囲で**過不足なくすべて**列挙した早見表を、印刷用A2横ポスターPDFとして生成するPythonスクリプトです。

[divider-poster](../divider-poster/) がE12/E24/E48を使って「1%刻みのグリッドに一番近い値」を当てはめるのに対し、こちらはE6限定で「実際に作れる分圧比」だけをそのまま並べます。E6は在庫に余りがちな系列なので、精度がそこまで要らない用途ではこちらで十分なことが多いです。

## 特徴

- E6系列（10 / 15 / 22 / 33 / 47 / 68、および10のべき乗倍）の抵抗2本で作れる分圧比を、1%〜50%の範囲で全数計算
- 目標値への近似ではなく、実在する組み合わせのみを掲載（本数は実行時に自動計算。デフォルトのE6系列では54件）
- 3列レイアウトで全件を1枚に収録

## 必要環境

- Python 3
- [reportlab](https://pypi.org/project/reportlab/)
- 日本語 TrueType フォント（IPAゴシック）

```bash
pip install -r requirements.txt
```

### フォントについて

このスクリプトはフォントパスを `/usr/share/fonts/opentype/ipafont-gothic/ipag.ttf` に**固定**で参照します（コマンドラインオプションでの変更には対応していません）。

```bash
# Ubuntu / Debian の例
sudo apt-get install fonts-ipafont-gothic
```

別環境・別フォントを使う場合は、`make_e6_poster.py` 冒頭の `pdfmetrics.registerFont(...)` の行を直接書き換えてください。

## 使い方

```bash
cd e6-divider-poster
python make_e6_poster.py
```

`e6_divider_poster.pdf`（A2横向き）がこのディレクトリ内に生成されます。データファイルはなく、E6系列の組み合わせをスクリプト実行時に毎回計算します。出力先やページサイズを変えるコマンドラインオプションはありません。

## 50%を超える比率が必要な場合

`Vout = Vin × R2/(R1+R2)` の性質上、目標比率が x%（x > 50）のときは (100-x)% の行を探し、その R1 と R2 を入れ替えれば厳密に x% になります（`R1/(R1+R2) = 1 - R2/(R1+R2)` という恒等式のため、近似ではなく正確に成立します）。

## データの計算方法

`make_e6_poster.py` 内の `find_all_ratios()` が、E6の基準値（10, 15, 22, 33, 47, 68）同士のすべての組み合わせに対して、R1側に -4〜+4桁の10のべき乗ずれを与えながら分圧比を計算し、1%〜50%に収まるものを抽出しています。同じ分圧比（小数点以下3桁に丸めて一致）が複数の組み合わせで実現できる場合は、桁ずれが小さく値も小さい、より扱いやすい組み合わせを採用しています。範囲や刻み方を変えたい場合はこの関数のパラメータ（`PCT_MIN` / `PCT_MAX` / `DECADE_SHIFTS`）を書き換えてください。
